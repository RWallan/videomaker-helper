# VideoMaker Helper

Micro tools to help content creators!


## Install

`pipx install https://github.com/dunossauro/videomaker-helper.git`


## Tools

### extract-audio

Extracts the audio from a video file.

Usage:

`vmh extract-audio <audio_path>`
